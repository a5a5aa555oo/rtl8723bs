### Linux Driver for RTL8723BS

Backporting the RTL8723BS driver to the old kernels from the linux git tree.

### How to use

1. make clean modules 
2. sudo make install

### Note

Tested on Debian 11.6 (kernel version: 5.10.0-21-amd64) and it works fine.
